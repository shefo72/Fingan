# Fingan ☕ – فنجان

### Arabic-Themed Food & Beverage Online Store

**Fingan** is an Arabic-themed food and beverage online store that combines traditional café culture with modern web design.  
The project focuses on delivering a smooth and engaging user experience through clean layouts, responsive design, and elegant animations.

This project was developed during the **SprintUp – Web Development Fundamentals** program by **Sprints**.

---

## 🚀 Features

- Arabic-inspired design and branding
- Responsive layout for all screen sizes
- Smooth scroll animations using **AOS (Animate On Scroll)**
- Clean and user-friendly interface
- Organized multi-section layout

---

## 🧱 Tech Stack

The project was built using:

- **HTML** – Structure and content
- **CSS** – Styling and layout
- **JavaScript** – Interactivity
- **AOS Library** – Scroll-based animations
